# DataMaster Website

Minimalistyczna, responsywna strona projektu DataMaster zbudowana wyłącznie w HTML, CSS i JavaScript.

## Uruchomienie

Otwórz `index.html` w przeglądarce.

## Pliki

- `index.html` — struktura strony
- `styles.css` — layout, responsywność i identyfikacja wizualna
- `script.js` — animacja bitów, reveal podczas scrollowania i menu mobilne

## Publikacja na GitHub Pages

1. Utwórz nowe repozytorium.
2. Wgraj wszystkie trzy pliki do katalogu głównego.
3. Wejdź w Settings → Pages.
4. Wybierz Deploy from a branch.
5. Wskaż gałąź `main` i katalog `/root`.

## Elementy do podmiany przed publikacją
